SAMOUSSA RUNNER

Un jeu savoureux et addictif !

Aidez le samoussa à éviter les obstacles et à collecter les épices dans ce jeu de type endless runner au style rétro. Courez à travers un décor inspiré de l'île de La Réunion, sautez par-dessus les obstacles et collectez les bonus pour augmenter votre score !

Atteignez les niveaux supérieurs pour gagner de véritables récompenses à échanger dans les restaurants WAYOWAYO !

Comment jouer

Sauter

Tapez sur l'écran pour faire sauter le samoussa par-dessus les obstacles.

Sauter plus haut

Maintenez votre doigt appuyé pour charger un saut plus puissant et aller plus haut.

Collecter les bonus

Ramassez les épices et les piments pour augmenter votre score. Attrapez la bière Dodo pour devenir temporairement invincible !

Système de récompenses

Jouez, progressez et gagnez de véritables récompenses à utiliser dans les restaurants WAYOWAYO ! Plus vous atteignez un niveau élevé, plus la récompense est importante.

Une fois le niveau atteint, un QR code unique est généré. Présentez-le dans n'importe quel restaurant WAYOWAYO pour recevoir votre cadeau !

Installez le jeu sur votre écran d'accueil

Pour une expérience optimale, installez Samoussa Runner sur l'écran d'accueil de votre smartphone.

1

Sur Android

Ouvrez le menu avec les trois points verticaux et sélectionnez "Ajouter à l'écran d'accueil"

2

Sur iOS

Appuyez sur l'icône de partage puis "Sur l'écran d'accueil"

3

Jouer hors ligne

Une fois installé, vous pourrez jouer même sans connexion internet !
